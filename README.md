## 快速开始

```bash
# 本地预览
python3 -m http.server 8080

# 检查 SSL 证书
python3 scripts/ssl-manager.py --check-remote

# 检查/更新时间戳
python3 scripts/timestamp-manager.py check
python3 scripts/timestamp-manager.py update
python3 scripts/timestamp-manager.py force
```
